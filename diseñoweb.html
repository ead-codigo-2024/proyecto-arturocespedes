Prácticas del Diseño

Diseño Web

El diseño web se centra en crear sitios intuitivos, estéticamente agradables y funcionales. Implica elegir la estructura adecuada, trabajar con tipografía, colores y elementos interactivos para asegurar una experiencia de usuario (UX) fluida. Es crucial que el diseño sea adaptable (responsive) y optimizado para distintos dispositivos.

UX/UI: La experiencia del usuario (UX) y la interfaz de usuario (UI) deben ser intuitivas y agradables.
Responsive Design: El diseño debe adaptarse a pantallas de distintos tamaños.

Ejercicios del contenido

Diseño de Página Web Básica

Investiga:
1.-Wireframe: Crea un esquema básico de la página (estructura)
2.-Diseño Visual: Usa Figma o Adobe XD para diseñar la interfaz con colores y tipografía adecuados.
3.-Exporta: Guarda el diseño como un archivo .png o .jpg.
